# Установка Visual Studio 2022

---

В рамках курса **Programming Foundations** в качестве основной среды разработки
будет использоваться **Visual Studio 2022**.

Visual Studio — это полноценная интегрированная среда разработки (IDE),
которая включает компилятор, отладчик и все необходимые инструменты
для работы с языком **C**.

---

## Шаг 1. Загрузка установщика

Перейдите по ссылке: https://aka.ms/vs/17/release/vs_community.exe

Скачайте **Visual Studio 2022 Community** — это бесплатная версия,
полностью подходящая для обучения и личных проектов.

---

## Шаг 2. Запуск установщика

После запуска установщика **Visual Studio Installer** появится окно
с информацией о подготовке к установке.

Для продолжения нажмите кнопку **«Продолжить»**.

![img.png](../tools%20materials/vs%202022/Установка%20visual%20studio%202022/img.png)

После этого установщик автоматически выполнит подготовку:
- загрузит необходимые компоненты;
- выполнит начальную установку Visual Studio Installer.

![img_2.png](../tools%20materials/vs%202022/Установка%20visual%20studio%202022/img_2.png)

Дождитесь завершения этапа подготовки.

---

## Шаг 3. Выбор рабочей нагрузки

После подготовки откроется окно выбора рабочих нагрузок.

В разделе **«Классические и мобильные приложения»**
необходимо выбрать:

- ✅ **Разработка классических приложений на C++**

![img_3.png](../tools%20materials/vs%202022/Установка%20visual%20studio%202022/img_3.png)

> ⚠️ Несмотря на название, именно эта рабочая нагрузка
> включает все необходимые инструменты для разработки на языке **C**.

Остальные рабочие нагрузки на данном этапе **выбирать не требуется**.

## Шаг 4. Проверка компонентов

После выбора рабочей нагрузки в правой части окна
отобразится список устанавливаемых компонентов.

Рекомендуется, чтобы были включены следующие пункты:

- **MSVC версии v143** (компилятор C/C++)
- **ATL-библиотека C++**
- **C++ Build Insights**
- **Отладчик**
- **Средства профилирования C++**
- **Средства CMake C++ для Windows**
- **Пакет SDK для Windows 10 или 11**

Остальные компоненты можно оставить включёнными по умолчанию.

---

## Шаг 5. Расположение установки

На вкладке **«Расположения установки»** можно оставить значения по умолчанию:

- путь установки Visual Studio;
- путь кэша загрузки;
- общие компоненты SDK.
- 
![img_4.png](../tools%20materials/vs%202022/Установка%20visual%20studio%202022/img_4.png)

Изменять эти параметры **необязательно**.

--- 

## Шаг 6. Запуск установки

После проверки параметров нажмите кнопку **«Установить»**.

Установщик начнёт загрузку и установку всех выбранных компонентов.
Процесс может занять некоторое время в зависимости от скорости интернета
и производительности системы.

Дождитесь полного завершения установки.

---

## Шаг 7. Завершение установки

После завершения установки Visual Studio будет готова к работе.
