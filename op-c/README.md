# Основы программирования (C)

Этот модуль посвящён базовым принципам программирования на языке C.

Мы начинаем с C, чтобы:
- понять, как работает программа на низком уровне;
- разобраться в базовых конструкциях языков программирования;
- заложить фундамент для перехода к C++ и алгоритмам.

Модуль будет постепенно дополняться новыми уроками и практикой.
