# Вход в Visual Studio 2022 через GitHub

---

Visual Studio 2022 позволяет выполнять вход не только через учётную запись Microsoft,
но и через **аккаунт GitHub**. Этот способ удобен, если вы планируете работать
с репозиториями и системой контроля версий.

> ⚠️ Для данного способа **обязательно наличие аккаунта GitHub**.  
> Если у вас его нет, воспользуйтесь гайдом  
> 👉 [Регистрация аккаунта GitHub](../Git%20и%20GitHub/Регистрация%20аккаунта%20GitHub.md)

---

## Экран входа в Visual Studio

При первом запуске Visual Studio отображается окно **«Вход в Visual Studio»**.

Доступны следующие варианты:
- вход с помощью учётной записи Microsoft;
- вход с помощью **учётных данных GitHub**;
- создание новой учётной записи;
- пропуск входа.

Для данного способа выберите пункт:

**«Войти с учётными данными GitHub»**.

## Авторизация через GitHub

После выбора входа через GitHub откроется окно авторизации GitHub
в браузере или встроенном окне.

1. Выберите аккаунт GitHub, который будет использоваться.
2. Нажмите кнопку **Continue**.
3. Подтвердите разрешение на доступ Visual Studio к аккаунту GitHub.

После успешной авторизации Visual Studio автоматически вернётся
в среду разработки.

## Персонализация Visual Studio

После входа Visual Studio предложит выполнить базовую настройку среды:

- выбрать профиль параметров разработки (можно оставить **Общие**);
- выбрать цветовую тему (светлая или тёмная).

Все эти параметры **не являются критичными** и могут быть изменены позже
в настройках Visual Studio.

Для продолжения нажмите **«Запустить Visual Studio»**.

## Стартовый экран Visual Studio

После завершения настройки откроется стартовый экран Visual Studio 2022.

Здесь доступны основные действия:
- **Клонирование репозитория** (GitHub, Azure DevOps и др.);
- **Открыть проект или решение**;
- **Открыть локальную папку**;
- **Создание проекта**.

На этом этапе Visual Studio полностью готова к работе.
