# Регистрация аккаунта GitHub

---

GitHub используется для хранения кода, работы с репозиториями
и обучения современным практикам разработки.
Для этого необходим аккаунт **GitHub**.

Ниже приведена пошаговая инструкция регистрации.

---

## Шаг 1. Переход на сайт GitHub

Перейдите на сайт:

👉 https://github.com

На главной странице нажмите кнопку **Sign up**.

![img.png](../%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82%D1%8B/git%20%26%20github/%D0%A0%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F%20%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B0%20GitHub/img.png)
---

## Шаг 2. Создание аккаунта

Откроется страница **Create your free account**.

Заполните следующие поля:

- **Email** — укажите действующий адрес электронной почты
- **Password** — задайте пароль
- **Username** — имя пользователя (будет использоваться в репозиториях)
- **Your Country/Region** — выберите страну
  > ⚠️ Рекомендуется **не выбирать Россию**

После заполнения нажмите кнопку **Create account**.

![img_1.png](../%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82%D1%8B/git%20%26%20github/%D0%A0%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F%20%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B0%20GitHub/img_1.png)
---

## Шаг 3. Проверка аккаунта

GitHub предложит пройти проверку безопасности.

Выберите вариант:
- **Visual puzzle**  
  или
- **Audio puzzle**

Пройдите проверку, чтобы продолжить регистрацию.

![img_2.png](../%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82%D1%8B/git%20%26%20github/%D0%A0%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F%20%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B0%20GitHub/img_2.png)
---

## Шаг 4. Подтверждение электронной почты

После проверки GitHub отправит код подтверждения
на указанный адрес электронной почты.

1. Откройте письмо от GitHub.
2. Введите код подтверждения на странице.
3. Нажмите **Continue**.

![img_3.png](../%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82%D1%8B/git%20%26%20github/%D0%A0%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F%20%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B0%20GitHub/img_3.png)
---

## Шаг 5. Вход в аккаунт

После успешного подтверждения появится сообщение
о создании аккаунта.

Введите:
- **Username или email**
- **Password**

Нажмите **Sign in**.

![img_4.png](../%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82%D1%8B/git%20%26%20github/%D0%A0%D0%B5%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D1%8F%20%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B0%20GitHub/img_4.png)
---

## Шаг 6. Главная страница GitHub

После входа откроется **Dashboard GitHub**.

С этого момента аккаунт полностью готов к работе.
Вы можете:
- создавать репозитории;
- работать с кодом;
- подключать GitHub к Visual Studio.
