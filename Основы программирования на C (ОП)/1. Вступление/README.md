# 1. Вступление

Модуль знакомит с языком программирования **C** и показывает, как выглядит первая программа.

Здесь не требуется глубокое знание синтаксиса. Главная цель — понять, что такое C, где он используется, как создать первую программу и что происходит с кодом до запуска.

## Материалы модуля

| № | Материал | Краткое описание |
|---|---|---|
| 1.1 | [Язык программирования C](<./1.1-Язык программирования C.md>) | Общее знакомство с языком C: история, особенности, области применения, компилируемая природа языка и роль Visual Studio 2022 в курсе. |
| 1.2 | [Первая программа на C в Visual Studio](<./1.2-Первая программа на C в Visual Studio.md>) | Создание первого проекта в Visual Studio 2022, добавление исходного файла, написание программы `Hello world`, запуск и первичный разбор кода. |
| 1.3 | [Чем отличается C от C++](<./1.3-Чем отличается C от C++.md>) | Краткое сравнение C и C++: процедурный подход, объектно-ориентированное программирование, работа с памятью, стандартные библиотеки и сферы применения. |
| 1.4 | [Как работает программа на C](<./1.4-Как работает программа на C.md>) | Объяснение общего пути программы: исходный код, компиляция, исполняемый файл и запуск. Также разбирается, что происходит при ошибках компиляции. |
