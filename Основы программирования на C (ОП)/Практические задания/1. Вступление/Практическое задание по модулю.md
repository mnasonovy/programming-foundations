# Практическое задание по модулю 1

## Тема

Первая программа на языке **C** в **Visual Studio 2022**.

---

## Цель

Создать, написать и запустить первую программу на языке C.

Теория для выполнения:

[1.2 — Первая программа на C в Visual Studio](<../../1. Вступление/1.2-Первая программа на C в Visual Studio.md>)

---

## Задание

Создайте новый проект в **Visual Studio 2022** и напишите программу, которая выводит сообщение в консоль.

Пример:

```text
Hello world!
```

Можно вывести своё сообщение, например:

```text
My first C program!
```

---

## Что нужно сделать

1. Создать проект в **Visual Studio 2022**.
2. Добавить файл с расширением `.c`.
3. Написать программу с `printf`.
4. Запустить программу.
5. Убедиться, что текст появился в консоли.

---

## Что нужно сдать

- исходный файл с программой;
- скриншот результата запуска программы.
