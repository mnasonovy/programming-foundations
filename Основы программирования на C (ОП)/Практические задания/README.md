# Практические задания

В этой папке находятся задания для самостоятельного выполнения по модулям курса.

Структура папки повторяет основные главы раздела:

```text
Практические задания
├── 1. Вступление
├── 2. Основы синтаксиса C
└── ...
```

Каждое задание содержит:

- тему;
- цель;
- список действий;
- требования к сдаче;
- критерии выполнения.
