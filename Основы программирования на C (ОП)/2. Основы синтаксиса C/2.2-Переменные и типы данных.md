# 2.2. Переменные и типы данных

В этом уроке разберём, как программа хранит данные.

В прошлом уроке мы изучили структуру программы на C. Теперь переходим к одной из самых важных тем в программировании — **переменным**.

---

## Цель урока

Понять:

- что такое переменная;
- зачем переменной нужен тип данных;
- как объявлять переменные;
- как записывать значения в переменные;
- какие базовые типы данных есть в C;
- как выводить значения переменных в консоль;
- какие имена переменных считаются хорошими.

---

## Что такое переменная

**Переменная** — это именованная область памяти, в которой программа хранит данные.

Проще говоря, переменную можно представить как коробку с названием:

```text
age → 18
```

Здесь:

- `age` — имя переменной;
- `18` — значение переменной.

Когда программа работает, она может:

- создать переменную;
- записать в неё значение;
- прочитать значение;
- изменить значение.

Например, если нужно хранить возраст пользователя, можно создать переменную `age`.

```c
int age = 18;
```

---

## Зачем нужны переменные

Без переменных программа могла бы только выводить заранее написанный текст.

Например:

```c
printf("Hello world!\n");
```

Но настоящие программы работают с данными:

- возраст;
- цена;
- оценка;
- имя;
- результат вычисления;
- количество товаров;
- координаты объекта;
- температура;
- скорость;
- пароль;
- номер попытки.

Для хранения таких данных и используются переменные.

---

## Объявление переменной

Перед использованием переменную нужно **объявить**.

Объявление сообщает компилятору:

```text
какое имя будет у переменной и какой тип данных она будет хранить
```

Пример:

```c
int age;
```

Эта строка означает:

```text
создать переменную age, которая хранит целое число
```

Разберём по частям:

| Часть | Значение |
|---|---|
| `int` | тип данных |
| `age` | имя переменной |
| `;` | конец инструкции |

---

## Присваивание значения

После объявления переменной в неё можно записать значение.

```c
age = 18;
```

Знак `=` называется **оператором присваивания**.

Он не означает математическое равенство. В программировании эта строка читается так:

```text
записать значение 18 в переменную age
```

Полный пример:

```c
#include <stdio.h>

int main(void)
{
    int age;

    age = 18;

    printf("Age: %d\n", age);

    return 0;
}
```

Результат:

```text
Age: 18
```

---

## Объявление с начальным значением

Чаще переменную объявляют и сразу задают ей значение.

```c
int age = 18;
```

Это короче и понятнее, чем писать в две строки:

```c
int age;
age = 18;
```

Оба варианта рабочие, но если значение известно заранее, лучше сразу задавать его при объявлении.

---

## Изменение значения переменной

Значение переменной можно менять.

```c
int age = 18;
age = 19;
```

После второй строки в переменной `age` будет храниться уже не `18`, а `19`.

Пример:

```c
#include <stdio.h>

int main(void)
{
    int age = 18;

    printf("Age: %d\n", age);

    age = 19;

    printf("New age: %d\n", age);

    return 0;
}
```

Результат:

```text
Age: 18
New age: 19
```

Важно:

> переменная хранит только последнее записанное значение.

---

## Тип данных

У каждой переменной в C есть **тип данных**.

Тип данных отвечает за две вещи:

1. какие значения можно хранить;
2. сколько памяти будет занимать переменная.

Например:

```c
int age = 18;
```

Тип `int` говорит, что переменная `age` хранит целое число.

А вот так можно хранить дробное число:

```c
double temperature = 36.6;
```

Тип `double` говорит, что переменная `temperature` хранит вещественное число.

---

## Почему в C нужно указывать тип

C — язык со **статической типизацией**.

Это значит, что тип переменной нужно указать заранее, ещё до запуска программы.

Например:

```c
int count = 10;
```

Компилятор сразу понимает:

```text
count — это целое число
```

Если попытаться использовать переменную не по назначению, компилятор может выдать предупреждение или ошибку.

Интересный момент:

> C даёт программисту много контроля, но требует аккуратности. Компилятор не будет всё делать за вас, поэтому важно понимать типы данных.

---

## Основные типы данных в C

На начальном этапе нам нужны четыре основных типа:

| Тип | Что хранит | Пример |
|---|---|---|
| `int` | целые числа | `10`, `-5`, `0` |
| `float` | дробные числа одинарной точности | `3.14f` |
| `double` | дробные числа двойной точности | `3.14` |
| `char` | один символ | `'A'`, `'7'`, `'+'` |

Разберём каждый тип отдельно.

---

## Тип `int`

`int` используется для хранения целых чисел.

Примеры:

```c
int age = 18;
int count = 10;
int temperature = -5;
```

Тип `int` подходит, если число не содержит дробной части.

Примеры данных для `int`:

- возраст;
- количество товаров;
- номер попытки;
- счёт в игре;
- количество студентов.

Пример программы:

```c
#include <stdio.h>

int main(void)
{
    int students = 25;

    printf("Students: %d\n", students);

    return 0;
}
```

Результат:

```text
Students: 25
```

---

## Тип `float`

`float` используется для хранения дробных чисел.

Пример:

```c
float price = 99.99f;
```

Обратите внимание на букву `f` в конце числа:

```c
99.99f
```

Она показывает, что это значение типа `float`.

Пример программы:

```c
#include <stdio.h>

int main(void)
{
    float price = 99.99f;

    printf("Price: %f\n", price);

    return 0;
}
```

Результат:

```text
Price: 99.989998
```

Почему может появиться не ровно `99.99`? Потому что дробные числа в компьютере часто хранятся приближённо.

Это нормально. Позже мы разберём, как красиво выводить дробные числа.

---

## Тип `double`

`double` тоже используется для дробных чисел, но обычно хранит их точнее, чем `float`.

Пример:

```c
double weight = 72.5;
```

Для большинства обычных вычислений с дробными числами в C часто используют именно `double`.

Пример:

```c
#include <stdio.h>

int main(void)
{
    double temperature = 36.6;

    printf("Temperature: %lf\n", temperature);

    return 0;
}
```

Результат:

```text
Temperature: 36.600000
```

---

## `float` или `double`

На начальном этапе можно запомнить так:

| Тип | Когда использовать |
|---|---|
| `float` | когда важна экономия памяти |
| `double` | когда нужна большая точность |

В учебных задачах, если нужно дробное число, чаще используйте `double`.

Интересный факт:

> название `double` происходит от “double precision”, то есть “двойная точность”. Исторически `double` занимал в памяти больше места, но позволял хранить дробные числа точнее.

---

## Тип `char`

`char` используется для хранения одного символа.

Пример:

```c
char grade = 'A';
```

Символ записывается в одинарных кавычках:

```c
'A'
```

Важно не путать:

```c
'A'   // символ
"A"   // строка
```

Пока просто запомните:

- один символ — одинарные кавычки;
- текст из нескольких символов — двойные кавычки.

Пример программы:

```c
#include <stdio.h>

int main(void)
{
    char grade = 'A';

    printf("Grade: %c\n", grade);

    return 0;
}
```

Результат:

```text
Grade: A
```

---

## Вывод переменных через `printf`

Чтобы вывести значение переменной, в `printf` используются специальные обозначения — **форматные спецификаторы**.

Пока достаточно знать основные:

| Тип | Спецификатор | Пример |
|---|---|---|
| `int` | `%d` | `printf("%d", age);` |
| `float` | `%f` | `printf("%f", price);` |
| `double` | `%lf` | `printf("%lf", weight);` |
| `char` | `%c` | `printf("%c", grade);` |

Пример:

```c
#include <stdio.h>

int main(void)
{
    int age = 18;
    double height = 180.5;
    char grade = 'A';

    printf("Age: %d\n", age);
    printf("Height: %lf\n", height);
    printf("Grade: %c\n", grade);

    return 0;
}
```

Результат:

```text
Age: 18
Height: 180.500000
Grade: A
```

Подробно `printf` будет разобран в следующем уроке. Сейчас главное понять: для каждого типа нужен свой способ вывода.

---

## Несколько переменных одного типа

Можно объявлять несколько переменных одного типа.

Так можно:

```c
int a = 10;
int b = 20;
int c = 30;
```

Также C позволяет писать в одну строку:

```c
int a = 10, b = 20, c = 30;
```

Но для новичка чаще лучше писать каждую переменную отдельно. Так код легче читать.

Хорошо:

```c
int age = 18;
int score = 100;
int attempts = 3;
```

Менее удобно для новичка:

```c
int age = 18, score = 100, attempts = 3;
```

---

## Имена переменных

Имя переменной должно быть понятным.

Плохой пример:

```c
int x = 18;
```

Лучше:

```c
int age = 18;
```

Плохой пример:

```c
int a = 5;
int b = 10;
```

Лучше:

```c
int width = 5;
int height = 10;
```

Хорошее имя переменной объясняет, что в ней хранится.

---

## Правила имён переменных

В C имя переменной может содержать:

- латинские буквы;
- цифры;
- символ подчёркивания `_`.

Но имя не может начинаться с цифры.

Правильно:

```c
int age;
int user_age;
int score2;
```

Неправильно:

```c
int 2score;
int user-age;
int user age;
```

Также нельзя использовать ключевые слова языка C:

```c
int return;
int int;
int char;
```

---

## Стиль именования

В учебных примерах лучше использовать стиль `snake_case`.

Пример:

```c
int user_age = 18;
double product_price = 149.99;
char user_grade = 'B';
```

`snake_case` — это стиль, где слова разделяются подчёркиванием.

Он хорошо читается и часто используется в C.

---

## Объявление без значения

Можно объявить переменную без начального значения:

```c
int age;
```

Но использовать такую переменную до присваивания значения нельзя.

Плохой пример:

```c
#include <stdio.h>

int main(void)
{
    int age;

    printf("Age: %d\n", age);

    return 0;
}
```

Проблема в том, что переменная `age` создана, но значение в неё не записано.

В ней может находиться случайное значение из памяти.

Правильно:

```c
#include <stdio.h>

int main(void)
{
    int age = 18;

    printf("Age: %d\n", age);

    return 0;
}
```

Правило:

> если значение известно — задавайте его сразу при объявлении.

---

## Полезный факт: память и размер типов

Разные типы данных занимают разное количество памяти.

Обычно:

| Тип | Примерный размер |
|---|---|
| `char` | 1 байт |
| `int` | 4 байта |
| `float` | 4 байта |
| `double` | 8 байт |

Точные размеры могут зависеть от компилятора и платформы.

Проверить размер типа можно с помощью `sizeof`:

```c
#include <stdio.h>

int main(void)
{
    printf("char: %zu\n", sizeof(char));
    printf("int: %zu\n", sizeof(int));
    printf("float: %zu\n", sizeof(float));
    printf("double: %zu\n", sizeof(double));

    return 0;
}
```

Эта программа выводит, сколько байт занимает каждый тип.

`sizeof` подробно разберём позже, но уже сейчас полезно знать: тип данных влияет не только на вид значения, но и на память.

## Частые ошибки новичков

### 1. Использование переменной без объявления

Ошибка:

```c
age = 18;
```

Правильно:

```c
int age = 18;
```

---

### 2. Использование переменной без значения

Плохо:

```c
int age;
printf("%d\n", age);
```

Лучше:

```c
int age = 18;
printf("%d\n", age);
```

---

### 3. Неправильный формат вывода

Ошибка:

```c
double weight = 72.5;
printf("%d\n", weight);
```

Правильно:

```c
double weight = 72.5;
printf("%lf\n", weight);
```

---

### 4. Неправильные кавычки для `char`

Ошибка:

```c
char grade = "A";
```

Правильно:

```c
char grade = 'A';
```

---

### 5. Непонятные имена переменных

Плохо:

```c
int a = 100;
int b = 5;
```

Лучше:

```c
int score = 100;
int attempts = 5;
```

## Практический пример

Напишем программу, которая хранит информацию о студенте.

```c
#include <stdio.h>

int main(void)
{
    int age = 20;
    double height = 178.5;
    char grade = 'A';

    printf("Student information\n");
    printf("Age: %d\n", age);
    printf("Height: %lf\n", height);
    printf("Grade: %c\n", grade);

    return 0;
}
```

Результат:

```text
Student information
Age: 20
Height: 178.500000
Grade: A
```

В этой программе:

| Переменная | Тип | Что хранит |
|---|---|---|
| `age` | `int` | возраст |
| `height` | `double` | рост |
| `grade` | `char` | оценку |

---

## Главное, что нужно запомнить

- Переменная хранит данные в программе.
- У каждой переменной есть имя, тип и значение.
- Тип данных определяет, какие значения можно хранить.
- `int` хранит целые числа.
- `float` и `double` хранят дробные числа.
- `char` хранит один символ.
- Переменную нужно объявить перед использованием.
- Если значение известно заранее, лучше задавать его сразу.
- Имя переменной должно быть понятным.
