# 2.1. Структура программы на C

В этом уроке разберём базовую структуру программы на языке **C**.

После первого модуля вы уже видели программу `Hello world`. Теперь нужно понять, из каких частей она состоит и за что отвечает каждая строка.

---

## Цель урока

Понять общий каркас программы на C:

- подключение библиотеки;
- функцию `main`;
- тело программы;
- инструкции;
- точку с запятой;
- команду `return 0`.

---

## Базовый пример программы

Рассмотрим простую программу:

```c
#include <stdio.h>

int main(void)
{
    printf("Hello world!\n");
    return 0;
}
```

Эта программа выводит в консоль сообщение:

```text
Hello world!
```

На первый взгляд кода мало, но в нём уже есть основные элементы любой программы на C.

---

## Общая схема программы

Простейшая программа на C обычно выглядит так:

```text
подключение библиотек

главная функция main
{
    инструкции программы;
    завершение программы;
}
```

Если записать ближе к реальному коду:

```c
#include <stdio.h>

int main(void)
{
    // здесь пишется код программы

    return 0;
}
```

---

## 1. Подключение библиотеки

Первая строка программы:

```c
#include <stdio.h>
```

`#include` подключает к программе внешний файл с готовыми возможностями.

В данном случае подключается файл:

```c
stdio.h
```

Он нужен для работы с вводом и выводом данных.

Например, функция `printf`, которая выводит текст в консоль, находится именно в этой библиотеке.

Без строки:

```c
#include <stdio.h>
```

компилятор может не понять, что такое `printf`.

---

## 2. Функция `main`

Следующая важная часть:

```c
int main(void)
```

`main` — это главная функция программы.

Именно с неё начинается выполнение программы на C.

Когда вы запускаете программу, компьютер ищет функцию `main` и начинает выполнять код внутри неё.

Разберём строку по частям:

| Часть | Значение |
|---|---|
| `int` | функция возвращает целое число |
| `main` | имя главной функции программы |
| `void` | функция не принимает входных параметров |

Пока не нужно глубоко разбирать функции. Главное запомнить:

> в программе на C должна быть функция `main`, потому что с неё начинается выполнение программы.

---

## 3. Фигурные скобки

После строки `int main(void)` идут фигурные скобки:

```c
{
    printf("Hello world!\n");
    return 0;
}
```

Фигурные скобки показывают, где начинается и где заканчивается тело функции.

Всё, что находится между `{` и `}`, относится к функции `main`.

```c
int main(void)
{
    // начало тела функции

    // здесь находятся инструкции

    // конец тела функции
}
```

Если забыть закрывающую скобку `}`, программа не скомпилируется.

---

## 4. Инструкции программы

Внутри функции находятся инструкции.

Инструкция — это отдельная команда, которую должна выполнить программа.

Например:

```c
printf("Hello world!\n");
```

Эта инструкция выводит текст в консоль.

Ещё одна инструкция:

```c
return 0;
```

Она завершает работу функции `main`.

Обычно инструкции выполняются сверху вниз:

```c
printf("Первая строка\n");
printf("Вторая строка\n");
printf("Третья строка\n");
```

Результат:

```text
Первая строка
Вторая строка
Третья строка
```

---

## 5. Точка с запятой

В языке C большинство инструкций заканчивается точкой с запятой:

```c
printf("Hello world!\n");
return 0;
```

Точка с запятой показывает компилятору, что команда закончилась.

Если забыть `;`, программа выдаст ошибку компиляции.

Неправильно:

```c
printf("Hello world!\n")
return 0;
```

Правильно:

```c
printf("Hello world!\n");
return 0;
```

---

## 6. Вывод текста через `printf`

Строка:

```c
printf("Hello world!\n");
```

выводит текст в консоль.

Разберём её:

| Часть | Значение |
|---|---|
| `printf` | функция вывода текста |
| `"Hello world!\n"` | строка, которую нужно вывести |
| `\n` | перенос строки |
| `;` | конец инструкции |

Символы внутри двойных кавычек выводятся на экран.

Например:

```c
printf("C is cool!\n");
```

Результат:

```text
C is cool!
```

---

## 7. Команда `return 0`

В конце программы находится строка:

```c
return 0;
```

Она завершает функцию `main`.

Число `0` обычно означает, что программа завершилась успешно.

Можно понимать это так:

```text
return 0; → программа завершилась нормально
```

На начальном этапе почти все программы будут заканчиваться этой строкой.

---

## Полный разбор примера

Ещё раз посмотрим на программу целиком:

```c
#include <stdio.h>

int main(void)
{
    printf("Hello world!\n");
    return 0;
}
```

Теперь построчно:

| Строка | Что делает |
|---|---|
| `#include <stdio.h>` | подключает библиотеку для ввода и вывода |
| `int main(void)` | объявляет главную функцию программы |
| `{` | начало тела функции |
| `printf("Hello world!\n");` | выводит текст в консоль |
| `return 0;` | завершает программу |
| `}` | конец тела функции |

---

## Минимальная программа на C

Минимальная рабочая программа может выглядеть так:

```c
int main(void)
{
    return 0;
}
```

Она ничего не выводит, но формально является программой.

Однако если нужно использовать `printf`, необходимо подключить `stdio.h`:

```c
#include <stdio.h>

int main(void)
{
    printf("Hello!\n");
    return 0;
}
```

## Частые ошибки новичков

### 1. Забыли точку с запятой

Ошибка:

```c
printf("Hello world!\n")
```

Правильно:

```c
printf("Hello world!\n");
```

---

### 2. Забыли закрыть фигурную скобку

Ошибка:

```c
int main(void)
{
    printf("Hello world!\n");
    return 0;
```

Правильно:

```c
int main(void)
{
    printf("Hello world!\n");
    return 0;
}
```

---

### 3. Написали `main` с ошибкой

Ошибка:

```c
int mian(void)
```

Правильно:

```c
int main(void)
```

Имя `main` должно быть написано правильно, потому что это точка входа в программу.

---

### 4. Забыли подключить `stdio.h`

Ошибка:

```c
int main(void)
{
    printf("Hello world!\n");
    return 0;
}
```

Правильно:

```c
#include <stdio.h>

int main(void)
{
    printf("Hello world!\n");
    return 0;
}
```

## Практический пример

Напишите программу, которая выводит несколько строк:

```c
#include <stdio.h>

int main(void)
{
    printf("My first program in C\n");
    printf("I am learning programming\n");
    printf("This is module 2\n");

    return 0;
}
```

Результат:

```text
My first program in C
I am learning programming
This is module 2
```

---

## Главное, что нужно запомнить

- Программа на C начинается с функции `main`.
- Код функции находится между фигурными скобками `{}`.
- Инструкции выполняются сверху вниз.
- Большинство инструкций заканчивается `;`.
- `#include <stdio.h>` нужен для использования `printf`.
- `printf` выводит текст в консоль.
- `return 0;` обычно означает успешное завершение программы.
