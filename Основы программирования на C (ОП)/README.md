# Основы программирования на C (ОП)

Раздел посвящён изучению основ программирования на языке **C**.

Цель раздела — пройти путь от первой программы до уверенного понимания базовых конструкций языка: переменных, условий, циклов, функций, массивов, указателей, структур и работы с файлами.

Этот раздел является фундаментом для дальнейшего изучения **C++**, объектно-ориентированного программирования, алгоритмов и структур данных.

## Как устроены главы

Обычно глава содержит:

- несколько теоретических уроков;
- примеры кода;
- краткие пояснения к важным моментам;
- типичные ошибки новичков;
- практические задания для закрепления.

Если в главе есть отдельный файл `README.md`, его стоит открыть первым.  
Он показывает, какие уроки входят в главу и что нужно понять после её прохождения.

---

## Как проходить раздел

Рекомендуемый порядок:

1. Открыть README главы.
2. Изучить уроки по порядку.
3. Запустить примеры кода самостоятельно.
4. Изменить значения в примерах и посмотреть результат.
5. Выполнить практическое задание.
6. Пройти тест по модулю, если он указан.
7. Сдать работу через Pull Request.

Главный принцип:

> сначала понять идею, потом закрепить её кодом.
